About the Project

Watershed Project provides accessible geospatial tools for analyzing and visualizing watershed data. Built on open standards, it aims to support environmental researchers, land managers, and anyone interested in understanding our water systems.

The project combines interactive web mapping with methodologies aligned to established frameworks from USFS, EPA, and state forestry standards.
